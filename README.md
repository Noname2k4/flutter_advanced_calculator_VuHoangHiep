# Advanced Calculator App

Ứng dụng máy tính đa chế độ được phát triển bằng Flutter, hỗ trợ nhiều chức năng tính toán từ cơ bản đến nâng cao.

## Tính năng chính

### Basic Mode
- Các phép tính cơ bản: cộng, trừ, nhân, chia
- Phần trăm (%)
- Đổi dấu (±)
- Xóa toàn bộ / xóa từng ký tự

### Scientific Mode
- Hàm lượng giác: sin, cos, tan
- Logarithm: ln, log
- Căn bậc hai (√)
- Lũy thừa (x², x^y)
- Hằng số Pi (π)
- Chuyển đổi DEG / RAD

### Programmer Mode
- Chuyển đổi số:
  - Binary
  - Octal
  - Decimal
  - Hexadecimal
- Phép toán bitwise:
  - AND
  - OR
  - XOR
  - NOT
